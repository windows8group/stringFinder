# stringFinder

\*Someone make a logo lol\*

A tool to find localizable strings and maybe, apply them to your code.

## Building

Requires Python 3.1x with uv the package manager.

\;\)

Note: You better run with `python` instead of `uv`:

```bash
$ python -m src
```
